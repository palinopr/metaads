## 🚀 How to Use the Enhanced Dashboard

1. **Install Dependencies**
   ```bash
   pnpm install
   ```

2. **Deploy Agents** (if not already done)
   ```bash
   pnpm deploy:agents
   ```

3. **Start Development Server**
   ```bash
   pnpm dev
   ```

4. **Run Tests**
   ```bash
   pnpm test
   ```

5. **Build for Production**
   ```bash
   pnpm build
   ```
